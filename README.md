
# **Book Review and Recommendation Platform**

## **Description**  
This project is a web-based application inspired by Goodreads, aimed at providing an engaging platform for book reviews and recommendations.  
The platform combines modern web technologies and advanced natural language processing (NLP) capabilities to deliver an enhanced user experience.

---

## **Technologies Used**  

### **Backend and API**  
- **FastAPI**: For building a high-performance backend REST API.  
- **NGINX**: For efficient server management and API routing.  
- **PostgreSQL**: To manage relational data storage effectively.  

### **Frontend**  
- **Angular**: For creating a dynamic and interactive user interface.  

### **Deployment**  
- **Docker**: For containerized deployment to ensure scalability and consistency.  

### **NLP Features**  
- Training the **LLaMA 3.2 1B** model (Large Language Model Meta AI, 3.2 Billion parameters) using **PyTorch** and **Python** to:  
  - Automatically summarize texts and book reviews.  
  - Provide concise and meaningful insights to enhance the platform's user experience.  

---

## **Functionalities**  

### **For Users**  
- Submit reviews for books and read reviews from other users.  
- View detailed book information and get personalized recommendations.  
- Access summarized reviews for quick insights.  

### **For Administrators**  
- Manage the book database (add, update, and delete books).  
- Monitor user activity and moderate content.  

---

## **Setup Instructions**  

### **Prerequisites**  
- Docker and Docker Compose installed.  
- Node.js and npm/yarn installed for the frontend setup.  
- PostgreSQL database configured and running.  

### **Steps to Run**  

#### **Backend**  
1. Clone the repository.  
2. Navigate to the backend directory:  
   ```bash  
   cd backend  
   ```  
3. Build and start the backend service using Docker:  
   ```bash  
   docker-compose up --build  
   ```  

#### **Frontend**  
1. Navigate to the frontend directory:  
   ```bash  
   cd frontend  
   ```  
2. Install dependencies:  
   ```bash  
   npm install  
   ```  
3. Start the development server:  
   ```bash  
   npm start  
   ```  

#### **NLP Features**  
1. Ensure the necessary Python environment is set up for training the **LLaMA 3.2 1B** model.  
2. Execute the training and deployment scripts provided in the `nlp` directory.
